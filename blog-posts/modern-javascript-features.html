Modern JavaScript Features Every Developer Should Know

March 1, 2025 8 min read JavaScript, ES6+, Web Development

Introduction

JavaScript has evolved significantly since the introduction of ES6 (ECMAScript 2015). In this post, we'll explore some of the most powerful features that every developer should be familiar with.

1. Destructuring

Destructuring allows you to extract values from objects and arrays into distinct variables:


// Object destructuring
const person = { name: 'John', age: 30 };
const { name, age } = person;

// Array destructuring
const numbers = [1, 2, 3];
const [first, second] = numbers;

2. Async/Await

Async/await provides a cleaner way to work with Promises:


async function fetchUserData() {
    try {
        const response = await fetch('https://api.example.com/user');
        const data = await response.json();
        return data;
    } catch (error) {
        console.error('Error:', error);
    }
}

3. Optional Chaining

Optional chaining helps prevent errors when accessing nested object properties:


const user = {
    details: {
        address: {
            street: '123 Main St'
        }
    }
};

// Without optional chaining
const street1 = user && user.details && user.details.address && user.details.address.street;

// With optional chaining
const street2 = user?.details?.address?.street;
